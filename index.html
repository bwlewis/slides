Bryan W. Lewis, September 2018

Why?

I forget mark down syntax details, and often find myself using HTML anyway.

I'm sick of bloated slides and often just want an easy, web-friendly slide environment.

HTML and CSS are pretty easy to use and give you

  • Easy lists like this
  • Fine control of page layout, look and feel
  • Minimal bloat (these slides are about 4k)
  • A geeky retro vibe

Using this template:

Wrap each slide in a "slide" div element like so: <div class="slide"> Your slide here.... </div>



...and place your slides in-between the <!--BEGIN SLIDES--> and <!--END OF SLIDES--> markers in this document.

You can put your slides on a web site or display them in a local web browser using the Python simple HTTP server:
cd (to your slide directory) python -m SimpleHTTPServer
  • Use the left/right arrows or the little buttons to navigate.
  • Use the 'p' key or print button to print the slides nicely.
  • Use the 'h' key to hide the buttons.

That's it!
